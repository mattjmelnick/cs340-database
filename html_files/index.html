Customers

The Customers page shows a table of the existing customers and can be used to add a new customer and update or delete an existing customer.

Sneakers

The Sneakers page shows a table of the sneaker inventory. Add a new sneaker and update or delete an existing sneaker.

Orders

The Orders page displays the order history table. Add order entries and update or delete existing orders.

Raffles

The Raffles page contains information about each raffle. Add a new raffle entry and update or delete an existing raffle.

Drawings

The Drawings page shows the date and day of each raffle. Add a new drawing entry and update or delete existing drawings.

Sneaker Orders

The Sneaker Orders page shows a table of the orders, which sneakers were purchased, and the quantity sold. Add a new transaction, and update or delete existing sneaker orders.

Raffle Orders

The Raffle Orders page shows a table of the orders and the raffle entries that were sold during the order. Add a new transaction, and update or delete existing raffle orders.

Customer Raffles

The Customer Raffles page contains information about each raffle, the customers who bought an entry, and each winner. Add a new customer raffle entry, and update or delete existing customer raffle entries.
